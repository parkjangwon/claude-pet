# ClaudePet
나만의 작고 귀여운 반려 클로드를 키워보세요!
 macOS에서만 작동합니다.
 손쉬운 사용 설정에 클로드펫을 활성화 해주세요! 비활성화 시 키보드 타이핑 기능을 사용할 수 없습니다.

간단 사용 설명!
 한번 클릭 시 : 클로드가 좋아합니다.
 우클릭 시(두 손가락 터치) : 메뉴를 엽니다.
 세게 누를 시 : 누른 방향의 반대 방향으로 클로드가 놀라 이동합니다! 사용 중인 프로그램에 클로드가 걸리적 거릴 시 세게 눌러 클로드를 이동하세요. (포스터치가 지원되는 맥북에서만 사용 할 수 있습니다.)

 타이핑을 하면 카운터가 올라요!(클로드 아래의 숫자) 클로드는 일정 시간마다 포만도가 떨어지며, 타이핑 카운터를 사용하여 포만도를 올릴 수 있어요. 포만도가 오르면 호감도도 같이 올라요! 호감도 레벨이 5단위로 올라갈 수록 친근한 말투로 대해줘요.
 클로드는 가끔 마우스의 위치로 따라와요.
